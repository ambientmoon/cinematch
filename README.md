# CineMatch 🎬

> 좋아하는 영화 한 편을 고르면, 닮은 영화 세 편을 찾아주는 웹 서비스

순수 HTML/CSS/JavaScript로 만든 정적 사이트입니다. 서버·로그인·API 키 없이 브라우저만으로 완전히 동작합니다.

## 무엇을 하는 프로젝트인가

영화 한 편을 입력하면, 미리 준비된 25편의 영화 데이터와 비교해 **장르·키워드가 가장 많이 겹치는 3편**을 추천합니다. 각 추천에는 "왜 추천하는지"가 한 문장으로 함께 표시됩니다.

## 사용 방법

1. 검색창에 영화 제목을 **영어로** 입력합니다 (예: `Inception`).
2. 자동완성 목록에서 영화를 하나 고릅니다.
3. 선택한 영화 정보와 함께 닮은 영화 3편이 추천됩니다.
4. `다른 영화로 다시 찾기` 버튼으로 처음부터 다시 검색할 수 있습니다.

## 사용 예시 ★

`Whiplash`(데이미언 셔젤, 음악 드라마)를 선택하면:

| 순위 | 추천 영화 | 추천 이유 |
|------|-----------|-----------|
| #1 | La La Land | Drama·Music 장르가 겹치고, jazz·ambition 키워드 공유 |
| #2 | Parasite | Drama 장르가 겹침 |
| #3 | Joker | Drama 장르가 겹침 |

`Spirited Away`(일본 애니메이션)를 선택하면 → Your Name, 반지의 제왕, Coco 순으로 추천됩니다.

## 입력 / 출력

| 입력 | 출력 |
|------|------|
| 영화 제목 (영어, 부분 검색 가능) | 선택 영화의 포스터·장르·감독·줄거리 + 닮은 영화 3편 + 추천 이유 |

## 추천 방식

- 선택한 영화와 나머지 24편을 각각 비교
- **장르 1개 겹칠 때마다 2점, 키워드 1개 겹칠 때마다 1점**
- 점수가 높은 순으로 상위 3편 선택 (동점이면 최신작 우선)
- 자기 자신은 추천에서 제외

## 아직 안 되는 것 / 한계 ★

직접 사용해보며(일부러 부숴보며) 확인한 한계입니다:

- **한글 제목 검색은 안 됨** — 검색은 영어 제목만 지원합니다 (예: `기생충` ❌ / `Parasite` ✅).
- **영화는 25편으로 고정** — TMDB API 대신 미리 담은 데이터를 쓰므로, 목록에 없는 영화는 검색되지 않습니다.
- **실제 영화 포스터 대신 색상 카드 사용** — 외부 이미지 서버(hotlink)가 환경에 따라 차단될 수 있어, 어디서나 안정적으로 보이도록 영화별 고유 색상+이니셜 카드를 자체 생성합니다.
- 빈 칸·이상한 값(이모지, 매우 긴 문자열) 입력 시 → 크래시 없이 "일치하는 영화가 없어요" 안내가 표시됩니다.
